# Module: Area Functions
This module contains functions that will find the area of a shape based on parameters given as input to the functions down below.

## Area of a Rectangle
![][area_rect]
```
rectangle(l, w);
```

## Area of a Parallelogram
![][area_para]
```
parallelogram(b, h);
```

## Area of a Triangle
![][area_trian]
```
triangle(b, h);
```

## Area of a Trapezoid
![][area_trapa]
```
trapezoid(h, b1, b2);
```

## Area of a Circle
![][area_circ]
```
circle(r);
```
