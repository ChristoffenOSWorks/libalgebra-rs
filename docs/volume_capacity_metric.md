# Module: Metric Volume/Capacity
Contains functions for volume/capacity in terms of the metric system.

## Liters to Milliliters
```
l_to_ml(l);
```

## Milliliters to Liters
```
ml_to_l(ml);
```

## Killoliters to Liters
```
kl_to_l(kl);
```

## Liters to Killoliters
```
l_to_kl(l);
```
