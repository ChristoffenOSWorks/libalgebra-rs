# Module: Metric Weight/Mass
Functions to convert between metric weight/mass

## Kilograms to Grams
```
kg_to_g(kg);
```

## Grams to Kilograms
```
g_to_kg(g);
```
## Grams to Milligrams
```
g_to_mg(g);
```

## Milligrams to Grams
```
mg_to_g(mg);
```
## Tons to Kilograms
```
t_to_kg(t);
```
## Kilograms to Tons
```
kg_to_t(kg);
```
