# Module: Customary Volume/Capacity
Functions to convert between customary volume and capacity.

## Gallons to Quarts
```
gal_to_qt();
```

## Quarts to Gallons
```
qt_to_gal();
```

## Gallons to Fluid Ounces
```
gal_to_floz();
```

## Fluid Ounces to Gallons
```
floz_to_gal();
```

## Quarts to Pints
```
qt_to_pt();
```

## Pints to Quarts
```
pt_to_qt();
```

## Pints to Cups
```
pt_to_c();
```

## Cups to Pints
```
c_to_pt();
```

## Cups to Fluid Ounces
```
c_to_floz();
```

## Fluid Ounces to Cups
```
floz_to_c();
```
